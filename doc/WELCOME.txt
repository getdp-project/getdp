GetDP is distributed under the terms of the GNU General Public License. See the
LICENSE.txt and CREDITS.txt files for more information.

The tutorials/ directory contains a step-by-step tutorial to learn GetDP.  The
reference manual is available here: http://getdp.info/doc/texinfo/.  For
additional examples see the ONELAB web site (http://onelab.info).

Notes:

* GetDP has no integrated graphical user interface: it is run from the command
  shell (terminal).

* If you are looking for a graphical user interface to GetDP, download Gmsh
  (http://gmsh.info). To launch GetDP via Gmsh, simply open a ".pro" file with
  Gmsh, which will automatically run GetDP as needed. Alternatively, go to the
  Gmsh solver module and click on the GetDP button. The OneLab web site
  (http://onelab.info) has more information.

* The problem definition files (".pro" files) are regular ASCII text files: you
  can create them with whatever text editor you prefer (VSCode, Notepad, Emacs,
  etc.).
